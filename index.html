My Work

  • Go-Chella

    A group organizing app centered around major events and conventions, Go-Chella is a full-stack project built using React/Redux on the frontend, and a backend of MongoDB, Node/Express.js, & Axios.

  • Kevin.Tv

    An homage to justin.tv, Kevin.Tv is a full-stack clone of the popular online streaming service Twitch.tv. The backend utilizes Ruby on Rails, a PostgreSQL database, & AJAX, while the frontend leverages React/Redux.

    A browser-based retool of the 1978 classic Space Invaders, built using Javascript and HTML5 Canvas. Levels scale in difficulty, and a leaderboard built utilizing Go and MongoDB is currently in the works.
